La veille informationnelle

Ma veille informationnelle porte sur React Native, un des principaux frameworks de développement mobile utilisés par les développeurs du monde entier aujourd’hui. Il offre des fonctionnalités robustes et un nombre important de fonctions puissantes pour les applications iOS et Android.

Qu'est ce qu'un framework ?

Concrètement, la traduction française de « framework » est « cadre de travail ». En d'autres termes, le framework, pour un développeur, c'est en quelque sorte, un modèle prêt à l'emploi, qu'il peut utiliser facilement en l'adaptant au projet sur lequel il travaille.

Qu'est ce que React Native ?

React Native fait référence à un cadre basé sur JavaScript utilisé pour créer des applications mobiles natives sur Android et la plate-forme iOS. Il est basé sur React, la bibliothèque JavaScript de Facebook, pour créer des interfaces utilisateur pour les plateformes mobiles.

Comment j'ai procédé ?

Pour effectuer la veille, j'ai principalement utilisé les outils que vous pouvez voir en haut (Feedly, Google et Reddit).
Grâce à Feedly ainsi que Google, j'ai pu apprendre beaucoup concernant React Native car au cas où vous ne le saviez pas, ces deux outils sont des agrégateurs de Flux RSS, c'est à dire qu'ils récupèrent le contenu disponible sur le web en rapport avec le sujet choisi et ils le font parvenir jusqu'à vous (via messagerie éléctronique par exemple).
Concernant Reddit, je m'en suis servi afin d'approfondir mes connaissances par rapport à mon sujet. Je m'y suis pris en lisant des posts de membres de la communauté de développeurs React Native, j'ai essayé d'intéragir, de discuter avec ces mêmes développeurs afin d'en apprendre plus sur eux ainsi que sur le framework (Mises à jour, avantages, principes d'utilisation).

Pourquoi avoir choisi React Native ?

J'ai choisi de faire ma veille informationnelle sur le React Native car cela concorde parfaitement avec mon projet professionnel, moi qui m'interresse de plus en plus au développement mobile. Mais ce n'est pas tout, j'ai notamment été accepté au sein de Sorbonne-Université pour y suivre l'année prochaine une licence professionnelle en développement web et mobile ou j'apprendrais à maitriser certains frameworks mobiles et web (front et back) tels que React Native, Flutter, Django, Node.js ou encore Symfony.
